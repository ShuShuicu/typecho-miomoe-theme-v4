#### 使用说明
1. 兼容所有PHP版本
2. 如果启用出现数据库查询错误请先将主题文件夹名称改为 <code>MioV4</code> 后保存一次主题设置。
3. 如果启用了 marked.js 解析器无法渲染文章内容请装一个 MarkdownParse 插件并开启开启 MathJax 支持(后续将写进主题)
